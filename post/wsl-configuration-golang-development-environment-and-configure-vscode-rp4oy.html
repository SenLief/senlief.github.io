WSL配置Golang开发环境并配置vscode

把开发环境都放在wsl上

  1. 安装golang
    1. 下载Golang
    2. 解压
      $ sudo tar -C /usr/local -xzf go1.17.1.linux-amd64.tar.gz
    3. 配置环境变量
$ vim .zshrc
export PATH="$PATH:/usr/local/go/bin"

$ sudo chmod +x /usr/local/go/bin
  1. 验证go是否安装
    $ go version
  2. 配置go环境变量
// 获取go相关变量
$ go env
// 设置go proxy
$ go env -w GOPROXY=https://goproxy.cn,direct
// 设置gopath
$ go env -w GOPATH=/home/zjan/.go
  1. 小打小闹vscode足以
    1. vscode安装go扩展
      1. 建议先连接到wsl，会安装wsl里面去。
    2. 安装go插件依赖包
      1. ctrl +shift +p 输入go install 勾选安装所有的包
  2. Enjoy！
